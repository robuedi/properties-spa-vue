## Properties SPA Project - Work in progress...

Technologies used in the project:

- Vue.js 3
- Pinia
- TypeScript
- Vue Router
- Tailwind CSS
- PrimeVue 

## How to use it

1. Install the Properties API project first and return here
2. Run `npm install`
3. Run `npm run dev`


### Web UI

Go to http://localhost:5173/
